Getting Started

Everything you'll need in order to start developing Minecraft mod using Minecraft Forge

Introduction

Hello I am Toby "Wehavecookies56" Wishart. Welcome to my site for Minecraft Forge Modding Tutorials! I am currently developing the Kingdom Keys Re:Coded mod for Minecraft and have decided to share some of my knowledge on Minecraft modding with Minecraft Forge.

These tutorials assume you are running Windows but mostly can be applied on any Operating System.

They also assume you have basic knowledge of using a computer and programming in Java.

Things you will need

Java Development Kit

The Java Development Kit (JDK) is an absolute requirement for developing anything with Java as Minecraft uses Java we require it to develop mods. It can be downloaded below. Make sure to check "Accept License Agreement" then download the version for your Operating System.

Download

Eclipse or IntelliJ IDEA

These are Intergrated Development Envrionments (IDE) which help tremendously when developing anything in Java or even other programming languages. They offer a much wider set of features than any simple text editor such as error highlighting, code suggestions, quick method overrides and much, much more.

In this tutorial I will be showing how to setup your mod workspace in both Eclipse and IntelliJ IDEA. Which one you choose is based on preference, I personally prefer IntelliJ IDEA to Eclipse however Eclipse is the easier of the two to setup.

Remember, you only need one of these! Don't install both unless you want to try both before deciding which you want to use.

Make sure to download "Eclipse IDE for Java Developers" either directly through the site or using the Eclipse Installer.

Download Eclipse

Make sure to download "Community".

Download IntelliJ IDEA

Minecraft Forge MDK

Minecraft Forge Mod Development Kit (MDK) are the tools required to setup Minecraft Forge for developing Mods

Make sure to download the "Mdk" for whichever version you want.

Download

Setup of this will be explained later on this page.

Git and a Github/Gitlab/Bitbucket account (Optional)

Git is a version control system which allows for easy collaboration and makes it much easier to keep track of changes and revert them. Github/Gitlab/Bitbucket intergrate Git so your repository is stored online.
